# Synonym Reference

Use this to build question stems that comply with the Synonym Rule.
Always seek the most natural paraphrase for the specific context — never mechanically substitute.

---

## Academic Verbs

| Source Text Word          | Question / Stem Synonym                            |
| ------------------------- | -------------------------------------------------- |
| demonstrate / show        | reveal · indicate · establish · illustrate         |
| examine / investigate     | explore · analyse · study · look into              |
| argue / claim / contend   | assert · propose · maintain · suggest              |
| conclude / find           | determine · establish · arrive at the view that    |
| increase / rise / grow    | expand · escalate · climb · surge                  |
| decrease / fall / decline | drop · reduce · diminish · contract                |
| cause / lead to           | result in · give rise to · contribute to · trigger |
| use / employ              | utilise · apply · make use of · deploy             |
| build / construct         | erect · develop · establish · create               |
| begin / start             | initiate · launch · commence · introduce           |

## Critical Reminder

"The building was **erected**" → natural.
"The policy was **erected**" → wrong.

Always read your paraphrase aloud and verify it means exactly what the original means in that context.
