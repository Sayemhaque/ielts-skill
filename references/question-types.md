# Question Type Master Reference

Apply these rules together with the Synonym Rule and Needle Rule at all times.

Each question type below shows its answer format template. Follow these exactly in your output.

---

## Multiple Choice (Single Answer)
- One correct answer; three distractors all mentioned somewhere in the source
- Tests inference and global understanding — not simple detail retrieval
- Stem must be a complete sentence or meaningful question
- All wrong options must come from real source information — never invented

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 35 | B | "the exact sentence that proves B is correct" |
```

## Multiple Choice (Multiple Answers)
- More than one correct answer; usually 2–3 out of 5–7 options
- All correct answers must be explicitly supported in the source
- Options may be listed A–E or A–G; state how many answers to select
- Distractors must be plausible and sourced from the text

## True / False / Not Given (Reading — Factual)
- Tests factual claims against the text
- **TRUE:** statement agrees with the text (using synonyms)
- **FALSE:** statement directly contradicts a specific line in the text
- **NOT GIVEN:** topic of the statement is simply never mentioned — no contradiction AND no confirmation
- ⚠ NEVER mark FALSE just because something is not mentioned. NOT GIVEN ≠ FALSE
- Reserved for Passage 1 and Passage 2 only

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 1 | FALSE | "the exact sentence that contradicts the statement" |
| 2 | TRUE | "the exact sentence that confirms the statement" |
| 3 | NOT GIVEN | "the passage does not mention [topic] at all — checked paragraphs A–E" |
```

## Yes / No / Not Given (Reading — Opinion)
- Tests the WRITER'S VIEW — not facts
- **YES:** statement agrees with writer's stated or clearly implied view
- **NO:** statement contradicts writer's view
- **NOT GIVEN:** writer expresses no view on this
- ONLY use in Passage 3. Never in Passage 1.

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 28 | NO | "the writer states '[exact quote]' which contradicts the statement" |
| 29 | YES | "the writer says '[exact quote]' — this directly supports the statement" |
```

## Matching Headings
- One heading maps to one paragraph
- Always more headings than paragraphs (at least 2 extras)
- Headings test the main idea — not a detail within the paragraph
- Reserved for Passage 2 and Passage 3

## Matching Headings
- One heading maps to one paragraph
- Always more headings than paragraphs (at least 2 extras)
- Headings test the main idea — not a detail within the paragraph
- Reserved for Passage 2 and Passage 3

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 14 | i | Paragraph A: "quote that shows this paragraph is about [topic]" |
| 15 | iv | Paragraph B: "quote that shows this paragraph describes [different topic]" |
```

## Matching Features
- Match people, organisations, or theories to their stated characteristics
- Word list must have MORE options than questions (at least 2 extras)
- Options may be used more than once — state this clearly
- Each matchable entity must be named and distinct

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 20 | C | "Professor X stated that '[exact quote]'" |
| 21 | A | "Dr Y argues that '[exact quote]'" |
```

## Matching (General)
- Used primarily for Listening Parts 2–3
- Match items from two sets based on script content
- Word list must have MORE options than questions (at least 2 extras)
- Each match must be explicitly stated in the source — no inferring

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 16 | B | "ground floor, west wing: 'to your left — that's the west wing'" |
| 17 | A | "ground floor, east wing: 'right at the back of the east wing'" |
```

## Matching Sentence Endings
- Complete a sentence by matching its beginning to the correct ending
- More endings than beginnings provided (at least 2 extras)
- Test understanding of cause-effect, chronology, and logical relationships
- Each completed sentence must be grammatically correct

## Sentence Completion
- Complete a sentence using words from the source
- ALWAYS state the word limit clearly: "NO MORE THAN TWO WORDS" or "ONE WORD ONLY"
- Answers must come verbatim from the source
- Completed sentence must be grammatically coherent
- Three words when limit is two = automatically wrong, even if correct information

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 26 | unsustainable | "the dominant transport model... is increasingly viewed as unsustainable" |
| 27 | cruising | "drivers spend significant time cruising between fares" |
```

## Note Completion
- Fill gaps in a set of notes organised under headings or subheadings
- ALWAYS state the word limit clearly: "NO MORE THAN TWO WORDS AND/OR A NUMBER"
- Answers must come verbatim from the source
- Notes follow the order of the source content

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 8 | larvae | "hatch into tiny larvae" |
| 9 | mulberry | "fed an exclusive diet of mulberry leaves" |
```

## Summary Completion (From Text)
- Fill gaps in a summary that paraphrases part of the source
- Words taken directly from the source text — not from a list
- ALWAYS state the word limit clearly
- Summary may cover one section or the whole passage

**Answer format:**
```
| Q | Answer | Needle |
|---|--------|--------|
| 38 | phi | "a measure denoted by the Greek letter phi" |
| 39 | complexity | "the complexity of neural responses... correlated strongly" |
```

## Summary Completion (From List)
- Fill gaps in a summary using words from a provided list
- Word list must have MORE options than gaps (at least 2 extras)
- Words in the list may be distractors from the source
- ALWAYS state the word limit clearly

## Table Completion
- Fill gaps in a table that organises source information
- ALWAYS state the word limit clearly
- Answers must come verbatim from the source
- Table tests ability to locate specific data and understand categories
