# Question Type Master Reference

Apply these rules together with the Synonym Rule and Needle Rule at all times.

---

## True / False / Not Given (Reading — Factual)
- Tests factual claims against the text
- **TRUE:** statement agrees with the text (using synonyms)
- **FALSE:** statement directly contradicts a specific line in the text
- **NOT GIVEN:** topic of the statement is simply never mentioned — no contradiction AND no confirmation
- ⚠ NEVER mark FALSE just because something is not mentioned. NOT GIVEN ≠ FALSE
- Reserved for Passage 1 and Passage 2 only

## Yes / No / Not Given (Reading — Opinion)
- Tests the WRITER'S VIEW — not facts
- **YES:** statement agrees with writer's stated or clearly implied view
- **NO:** statement contradicts writer's view
- **NOT GIVEN:** writer expresses no view on this
- ONLY use in Passage 3. Never in Passage 1.

## Matching Information to Paragraphs
- One piece of information maps to one paragraph (A, B, C…)
- Some paragraphs may be used more than once; some may not be used — state this in instructions
- Test specific, locatable content — not a paragraph's general theme

## Matching Features
- Match people, organisations, or theories to their stated characteristics
- Word list must have MORE options than questions (at least 2 extras)
- Options may be used more than once — state this clearly
- Each matchable entity must be named and distinct

## Summary / Note / Sentence Completion
- ALWAYS state the word limit clearly: "NO MORE THAN TWO WORDS" or "ONE WORD ONLY"
- Answers must come verbatim from the source (reading) or script (listening)
- Completed text must be grammatically coherent
- Three words when limit is two = automatically wrong, even if correct information

## Multiple Choice (4 options: A, B, C, D)
- One correct answer; three distractors all mentioned somewhere in the source
- Tests inference and global understanding — not simple detail retrieval
- Stem must be a complete sentence or meaningful question
- All wrong options must come from real source information — never invented

## Matching Headings
- One heading maps to one paragraph
- Always more headings than paragraphs (at least 2 extras)
- Headings test the main idea — not a detail within the paragraph
- Reserved for Passage 2 and Passage 3

## Diagram / Map / Flow-Chart Labelling
- Word list must have MORE options than gaps (at least 2 extras)
- Every label provable by a specific line in the text or moment in the script
- Visual must be entirely consistent with every description in its source
- For Listening Part 2 maps: complete Spatial Mapping Grid before drawing the map
