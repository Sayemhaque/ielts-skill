# Spatial Mapping Grid Tool

**Mandatory for every Listening Part 2 map question.**
Complete BEFORE writing the script, drawing the map, or creating any questions.

---

## Why This Exists

A student who follows the script correctly but gets a wrong answer because the map contradicts the script has been failed by the content creator, not by their own ability.

---

## How to Use It

For every unlabelled location, commit to all four things before any other work:

1. **Name** — what it will be called in the word list and on the map
2. **Grid position** — where it physically sits (use a 3×3 or 4×4 coordinate grid)
3. **Exact spoken cue** — the precise sentence the speaker uses to describe its location
4. **Reference anchor** — which pre-labelled location is the navigation starting point

When all 5 unlabelled locations are filled in, read every row in sequence and ask:
_"Can I draw this without contradiction?"_
If yes → proceed. If no → rewrite the offending cue first.

---

## Required Spatial Prepositions

Use at least 5 of these in every Part 2 script:

- adjacent to
- opposite
- tucked away
- to the north / south / east / west of
- directly behind
- between X and Y
- facing
- at the far end of
- just to the left / right of
- in the corner of

---

## Common Failures

| Failure             | Example                                                          | Fix                                  |
| ------------------- | ---------------------------------------------------------------- | ------------------------------------ |
| Contradiction       | Script says "opposite the café" but map places them side by side | Rewrite cue or move location         |
| Orphaned anchor     | Script references "the north exit" but it's not pre-labelled     | Add as anchor or remove reference    |
| Impossible walk     | Route doubles back disorienting                                  | Redesign tour to flow logically      |
| Duplicate position  | Two locations in same grid cell                                  | Assign distinct positions            |
| Decoy too similar   | "Conference Room" vs "Meeting Room" in word list                 | Replace with clearly distinct option |
| Missing preposition | Location described by function only, no spatial cue              | Add explicit spatial preposition     |
