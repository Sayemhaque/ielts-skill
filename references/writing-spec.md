# Writing Module Specification

## Overview
- 2 tasks, 60 minutes total
- Task 2 carries double the marks of Task 1
- Model answers required: Band 6, Band 7, Band 8 for every task

---

## Task 1 — Academic Report Writing

| Parameter | Specification |
|---|---|
| Minimum Words | 150 words |
| Recommended Time | 20 minutes |
| What Students Must Do | Describe and summarise key features; make comparisons |
| What Students Must NOT Do | Explain why / give opinions / make recommendations |

**Visual types — rotate across tests:**

| Visual Type | Key Testing Focus |
|---|---|
| Bar / Column Chart | Comparing categories across groups or time points |
| Line Graph | Describing trends over time (rise, fall, fluctuate, plateau) |
| Pie Chart | Describing proportions and largest/smallest segments |
| Table | Selecting the most significant data points |
| Process Diagram | Describing a sequence of 5–8 stages with arrows |
| Map Comparison | Describing changes between two time periods ("before" and "after") |

**Visual design rules:**
- Always include a clear title and labelled axes / legend
- Include at least one clear trend and one point of comparison
- Process diagrams: 5–8 stages with clear arrows
- Maps: show "before" and "after" with notable structural changes

**Writing rules:**
- Model answer MUST open with an overview sentence — never copied from the title
- Data must be cited precisely: not "it went up" but "it rose by 15%"
- No bullet points in any model answer — continuous prose only
- Tense must match the visual: past tense for historical data, present for static visuals

**Band descriptors:**

| Band | Key Characteristics |
|---|---|
| 8–9 | All key features covered; accurate data cited; clear overview; sophisticated cohesion |
| 7 | Key features covered with minor omissions; clear overview; good cohesion |
| 6 | Main features covered but some inaccuracy; overview present but underdeveloped |
| 5 | Key features described but data may be inaccurate; no clear overview |

## Task 2 — Academic Essay Writing

| Parameter | Specification |
|---|---|
| Minimum Words | 250 words |
| Recommended Time | 40 minutes |
| Format | Discursive essay responding to a statement or question |

**Essay types — rotate across tests:**

| Type | Prompt Pattern | Required Structure |
|---|---|---|
| Opinion / Argument | "To what extent do you agree or disagree?" | Clear position + 2 supporting arguments + concession |
| Discussion | "Discuss both views and give your opinion" | View A + View B + writer's own view |
| Problem / Solution | "What are the causes? What solutions can you suggest?" | Causes paragraph + Solutions paragraph |
| Two-Part Question | "Why is this happening? Is it a positive or negative development?" | Answer part 1 fully + Answer part 2 fully |
| Advantage / Disadvantage | "Do the advantages outweigh the disadvantages?" | Advantages + Disadvantages + overall judgement |

**Prompt writing rules:**
- Every prompt must have a clear, debatable position — no obvious single answer
- Use real-world relevance
- Never longer than 3 sentences
- Always end with: "To what extent do you agree or disagree? Give reasons for your answer and include any relevant examples from your own knowledge or experience."

**Model answer requirements:**
- 3 model answers per task: Band 6, Band 7, Band 8
- Each annotated — highlight what makes it that band
- Examiner commentary paragraph below each answer
- No bullet points — continuous prose only
- Band 8 must include: nominalisation + concession structure + advanced cohesive device

**Band descriptors:**

| Band | Task Achievement | Coherence | Lexis | Grammar |
|---|---|---|---|---|
| 8–9 | Fully addresses all parts; position clear and developed | Seamless cohesion; varied paragraphing | Wide, precise; rare errors | Wide range; errors are rare |
| 7 | All parts addressed; position clear | Clear progression; good cohesive devices | Good range; some awkwardness | Varied structures; some errors |
| 6 | Main parts addressed; position present | Reasonable organisation; some cohesion errors | Adequate range; some repetition | Mix of simple and complex; errors noticeable |
| 5 | Partially addresses prompt; position unclear | Some organisation; overuse of basic connectors | Limited range; frequent repetition | Limited range; frequent errors |

## QA Checklist — Run Before Output

- [ ] Task 1 visual has a clear title and labelled axes/legend
- [ ] Task 1 visual contains at least one clear trend and one comparison point
- [ ] Task 1 model answer opens with an overview sentence (not copied from title)
- [ ] Task 1 model answer cites specific data accurately
- [ ] Task 2 prompt clearly states the essay type
- [ ] Task 2 prompt ends with a complete instruction sentence
- [ ] Task 2 prompt is debatable — no obvious single answer
- [ ] Task 2 prompt is 3 sentences or fewer
- [ ] Three model answers provided per task (Band 6, Band 7, Band 8)
- [ ] All model answers are in continuous prose — no bullet points
- [ ] Examiner commentary included for each model answer
- [ ] Band 8 answer includes nominalisation, a concession, and an advanced cohesive device
